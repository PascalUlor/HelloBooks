    Introduction

    Welcome to the Hello Books API! You can use our API to access the Hello Books API endpoints which can get information on various books. HelloBooks is an application that helps manage a library and its processes like stocking, tracking and borrowing books to individuals.

    The API can be consumed using any programming language as it returns JSON for all API calls. You can view the coding examples in the dark area to the right

    Features

    This API has the following features:

    Unauthenticated Users

    • Creating an account
    • Signing in as an existing user
    • View Recent Books
    • Get User Levels list
    • View Book details

    Authenticated Users

    • Change password
    • View all books
    • Search for a book from all books
    • Borrow and return borrowed books
    • View borrowing history
    • Receive email notification when a surcharge is place for late return of borrowed book

    Admin

    • Add, edit and delete books
    • Add, edit and delete categories
    • Group books by their respective categories
    • Edit user levels
    • View borrowing and return notifications
    • Retrieve a user

    Change Password

    This endpoint handles change password request by authenticated users,

    • Users who created account using google login cannot change password.

    HTTP Request

    onSuccess - Status code (200)

    onError - Status code (400, 409, 404, 500)

    "oldPassword": "testuser",
    "password": "testuser",

    Errors

    This error section is stored in a separate file in `includes/_errors.md`.

    The Hello Books API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- You made a bad request.
    401 Unauthorized -- You are not unauthorized to view this route.
    403 Forbidden -- You are not allowed to view this route.
    404 Not Found -- This resource can't be found.
    409 Conflict -- This resource already exists
    422 Unprocessable Entity - This resource is invalid
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
