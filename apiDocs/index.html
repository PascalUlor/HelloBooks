    Introduction

    Welcome to the Hello Books API! You can use our API to access the Hello Books API endpoints which can get information on various books.HelloBooks is an application that helps manage a library and its processes like stocking, tracking and borrowing books to individuals.

    We have language bindings in Javascript! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    This example API documentation page was created with Slate. Feel free to edit it and use it as a base for your own API's documentation.

    Users

    Register User

    Request:

    {
  "username":"andela"
  "password":"ebelehigh"
  "firstname":"Benny"
  "passwordConfirmation":"ebelehigh"

    • Endpoint: POST: /api/v1/users/auth/signup
    • Body (application/json)

    • Status: 201: Created
    • Body (application/json)

    Books

    Create Books

    Request:

    {
  "title":"The Litigators"
  "author":"John Grisham"
  "category":"Fiction"
  "quantity":20
  "description":"This is a test book"
  "bookimage":"http://res.cloudinary.com/digpnxufx/image/upload/v1506272697/book3_cmhfot.jpg"
}

    Response:

    {
    "message": "The Litigators has been added to the library"
}

    Request

    • Endpoint: POST: /api/v1/books
    • Requires: Authorization
    • Body (application/json)

    Response

    • Status: 201: OK
    • Body (application/json)
    Authenticated users can access this route.

    Errors

    This error section is stored in a separate file in `includes/_errors.md`.

    The Hello Books API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- You made a bad request.
    401 Unauthorized -- You are not unauthorized to view this route.
    403 Forbidden -- You are not allowed to view this route.
    404 Not Found -- This resource can't be found.
    409 Conflict -- This resource already exists
    422 Unprocessable Entity - This resource is invalid
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
